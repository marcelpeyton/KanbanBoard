## Installation
Clone this repo then:
```npm i```
The code is reliant on an .env file with
```
DB_NAME=
DB_USER=
DB_PASSWORD=
JWT_SECRET_KEY=
```
Place the .env in at the root of the project

## Usage
To use:
```npm start:dev```
Or go to this website https://candidate-search-85yq.onrender.com/
